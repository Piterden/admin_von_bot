  - [ ] Увеличение/уменьшение репутации при ответе на сообщение

## Контрибуции:

```bash
$ git clone git@github.com:Piterden/admin_von_bot.git
$ cd admin_von_bot
$ npm i
$ cp .env.example .env
$ editor .env
$ npm run migrate
```
